### 🎨 AI PPT Redesign Tool
這是一個基於 Streamlit 開發的智慧簡報重新設計工具。本專案展示了如何利用 AI 邏輯與 Python 自動化處理技術，將一份結構簡單的 PPT 轉化為具有設計感的專業簡報。

### 🌟 專案亮點
本工具解決了「內容已有，但排版太醜」的痛點。透過解析原始投影片的語義結構，系統會自動重新套用視覺規範，並產出兩種風格的設計方案供使用者選擇。

### 🚀 核心功能

一鍵內容提取：自動辨識投影片標題與本文內容，支援多頁處理。

雙風格重新設計：

Minimalist (極簡主義)：以大面積留白、高品質字體間距為核心，適合商務提案。

Tech-Future (科技未來)：採用深色模式與發光色調，適合技術分享與前瞻報告。

雲端即時渲染：無須安裝 PowerPoint，直接在瀏覽器完成轉換並下載檔案。

### 🛠️ 技術框架

UI 框架：Streamlit - 提供流暢的 Web 互動介面。

簡報處理引擎：python-pptx - 負責 PPTX 檔案的底層結構解析與物件重建。

設計邏輯：使用 RGB 色彩矩陣 定義視覺調性。自動化 Placeholder 佈局映射，確保內容位置精準。

### 📖 使用說明
上傳檔案：將你的 .pptx 簡報拖入上傳區塊。AI 解析：系統會掃描每一張投影片的標題與條列式文字。預覽與下載：介面會顯示解析出的內容預覽。點擊 「下載極簡風格」 或 「下載科技風格」 即可獲得重新設計後的檔案。

### 📋 需求環境若要於本地端運行此 Demo，請安裝以下依賴套件：
```Bash
pip install streamlit python-pptx
```
並執行：
```Bash
streamlit run app.py
```
### 🧠 AI 設計邏輯參考
本專案的設計思路參考了 AI 輔助設計的標準流程：「內容(Content) 與 表現(Presentation) 分離」

解構：從舊版型中抽離純文字資訊。

風格化：套用預設的視覺參數（如標題字重、背景色 $RGB$、強調色）。

重構：將資訊填入全新的 XML 結構中，生成高品質的 PPTX 檔案。
